# Festivite - Gerador de Convites

Este projeto faz parte de uma trilha de aprendizado em Desenvolvimento Full-Stack. Focado no desenvolvimento de uma interface de formulário simples para a criação de convites digitais. O objetivo principal foi exercitar a estruturação semântica do HTML e a estilização avançada utilizando apenas recursos nativos do CSS. Ainda não foram aplicadas técnicas de responsividade, funcionando apenas para computadores.

## Tecnologias e Conceitos Aplicados

* **HTML5 Semântico**: Uso rigoroso de `fieldset`, `legend`, `label` e tipos de inputs específicos para garantir acessibilidade e organização.
* **CSS Grid & Flexbox**: Utilizados para criar um layout bidimensional robusto, permitindo o alinhamento preciso de elementos como o seletor de temas e o grid de cores.
* **Nesting CSS**: Aplicação da nova sintaxe nativa de aninhamento do CSS para manter as folhas de estilo organizadas e fáceis de ler.
* **CSS Custom Properties (Variáveis)**: Criação de um design system simplificado com tokens de cores, fontes e espaçamentos para facilitar a manutenção.
* **Semantic Commits**: Padronização de mensagens de commit para manter um histórico de projeto profissional e organizado.
